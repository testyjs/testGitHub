github 是一个代码托管中心 1、本地库初始化 命令：git add 效果：在执行命令的文件夹中生成一个.git文件夹，其中config是配置文件 2、设置签名（用来区分不同开发人员的身份） 用户名： email地址： 辨析：这里设置的签名和远程库的账号密码没有任何关系 命令：git config 1、项目级别：仅在当前本地库范围内有效 git config user.name yin_pro git config user.email 568829266@qq.com 设置的用户名和email地址在.git/config文件中 2、系统用户级别：登录当前操作系统的用户范围内有效 git config --global user.name yin git config --global user.email 568829266@qq.com 系统级别的设置在~/.gitconfig文件中 3、优先级：项目级别 > 系统用户级别，二者都没有是不允许的
